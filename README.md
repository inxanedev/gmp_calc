# gmp_calc
GMP based calculator made in C++ for linux

# Commands
    
    - add x  => addition
    - sub x  => subtraction
    - mul x  => multiplication
    - div x  => division
    - pow x  => exponentiation
    -   abs  => absolute value
    - prec x => set precision to x bits
    -    clr => clear screen
    -   fact => factorial (strips fractional part and applies abs !!!)
